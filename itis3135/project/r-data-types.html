Data Types in R

Common Types

R has several basic data types:

  • numeric – 3.14, 100, -5
  • character – "hello", "R", "stats"
  • logical – TRUE, FALSE

You can check a value’s type using class(x).

Example

x <- 42
class(x)       # "numeric"

name <- "Ben"
class(name)    # "character"

Atomic vs Complex Types

In R, many values are built from basic atomic types (like numeric or character). These values can then be organized into more complex structures such as vectors, lists, or data frames.

Category Type Examples How it is used
Atomic numeric 3.14, 100, -5 Store numbers for calculations.
Atomic character "hello", "R" Store text labels, names, or categories.
Atomic logical TRUE, FALSE Store yes/no decisions or test results.
Complex vector c(1, 2, 3) A sequence of values of the same atomic type.
Complex list list(1, "a", TRUE) Collection of values that can be different types.
Complex data frame data.frame(name, age) Table of columns (like a spreadsheet) used for data analysis.
