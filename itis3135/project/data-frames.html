Data Frames

What is a Data Frame?

A data frame is like a table: rows are records, columns are variables.

Data frames are the main way R stores real-world data.

Creating a Data Frame

people <- data.frame(
  name = c("Ben", "Sara", "Leo"),
  age = c(21, 22, 20),
  major = c("Stats", "CS", "Math")
)

You can view the data with head(people) or View(people) inside RStudio.
