Vectors in R

What is a Vector?

A vector is an ordered collection of values of the same type.

Create one using c() (combine):

nums <- c(10, 20, 30, 40)
words <- c("a", "b", "c")

Accessing Elements

You can access items by position (index):

nums[1]   # 10
nums[3]   # 30

Vectorized Math

In R, many operations are vectorized. That means a single operation is applied to every element of a vector automatically. 

nums <- c(10, 20, 30)
nums * 2   # 20  40  60

Here, R multiplies each element of nums by 2. You do not need a loop to get the new values.
